Scroll-Wiz

Intro

Scroll-Wiz is a micro library, created as solution to facilitate the implentation of controlled animated elements (front-end webdev).The animation's implentation mainly happens in your css code.

Each animated element takes a minimum of 2 CSS classes, a main class and an animation class.
The main class should have the properties for the element outside of the animation range. For example, if the result for a specific element is meant to take place between scroll values (30 - 300), you should include in this class the properties for the element before scrolling to 30, and after scrolling past 300. It should as well include the transition values. This class should be added to the element in html.
The animation class , which includes the properties that the element will have within the animation range. You need to declare this class in the CSS file, but no need to add it to the element in html, since this class will be added programatically whenever the element is within the animation range.
Each animation class can be used several times, at several animation ranges, or you can use a separate animation class for each animation.

Javascript code in the file (scrollwiz/sw.js) controls adding and removing classes at a specific vertical scroll value, and keeps track of the scroll-state.
